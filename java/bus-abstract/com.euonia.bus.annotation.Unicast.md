# Unicast

> 标记消息为单播消息，表示消息只会发送给一个订阅者。
> 总线会确保仅有一个订阅者接收该消息。

- **Type**: annotation
- **Package**: `com.euonia.bus.annotation`
- **Target**: `@Target({ElementType.TYPE})` — 可应用于类/接口
- **Retention**: `@Retention(RetentionPolicy.RUNTIME)` — 运行时保留
- **Author**: damon(zhaorong@outlook.com)

## Elements

> 此注解不包含任何元素，仅作为标记注解使用。
