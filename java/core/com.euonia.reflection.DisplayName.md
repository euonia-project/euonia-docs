# DisplayName

> 用于为字段指定显示名称的注解，可在 UI 或错误消息中使用以替代实际的字段名。

- **Type**: annotation
- **Package**: `com.euonia.reflection`
- **Author**: damon(zhaorong@outlook.com)

## Elements

| Element | Type | Default | Description |
|---------|------|---------|-------------|
| `value` | `String` | *(required)* | 显示名称的值 |
